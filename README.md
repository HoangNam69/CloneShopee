# CloneShopee

# Dựng base và kiểm tra kết nối giữa các file css với file index.html
    Đầu tiên source được dựng với cây thư mục như sau
        assets/
            css/
                main.css
                base.css
        index.html
    Run live server -> mở dev tool của trình duyệt vào source để kiểm tra hai file css đã được link thành công chưa
    
# Reset CSS
    Có 2 cách để format lại pages:
        + Format bằng code css thủ công
        + Format bằng cách gọi cnd normalize css (Áp dụng các này)
    Reset css bằng cách 2 ta sẽ copy cnd min.css và sử dụng tag link vào index.html paste đường dẫn vào href là thành công
